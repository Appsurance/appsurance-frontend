# Appsurance Technologies - Enterprise Website

A modern, professional website for Appsurance Technologies built with React.

## Features

- Clean, enterprise-grade design
- Responsive layout
- Multi-page navigation (Home, About, Services, Contact)
- Professional color scheme and typography
- Smooth animations and transitions

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Technology Stack

- React 18
- React Router DOM v6
- CSS3 with custom properties
- Inter font family
